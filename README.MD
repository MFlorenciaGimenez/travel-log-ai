# Travel-Log AI Microservice

An AI-powered microservice built with FastAPI that enables semantic search and context-aware question answering using Retrieval-Augmented Generation (RAG).

This service is designed to be consumed by any backend (such as a NestJS API) and acts as the AI layer of a larger application.
